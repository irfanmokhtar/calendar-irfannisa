# Irfan & Nisa — shared calendar

Mobile-first PWA shared calendar for two. Vanilla HTML/CSS/JS, no build step.

- Upcoming activities first, month calendar below
- Activities tagged Irfan / Nisa / Us two (color-coded)
- Data lives in localStorage on each phone; synced through a shared JSON store (npoint.io)
- Installable: open in phone browser → Add to Home Screen

Deployed on GitHub Pages.
